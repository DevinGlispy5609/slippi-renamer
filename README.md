# Slippi Renamer

Rename Slippi (.slp) game files to include characters, tags, and stage.

## Installation

```
npm install
```

## Usage

Given `*.slp` files as input, it will rename the files from:

**Singles:**
```
Game_20190530T222709.slp -> 20190530T222709 - Marth (BRY) vs Fox - Battlefield.slp
```

**Doubles:**
```
Game_20190521T212659.slp -> 20190521T212659 - Falco & Marth vs Fox (DJ) & Fox (SWRV) - Yoshi's Story.slp
```
